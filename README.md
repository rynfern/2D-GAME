# The Queen’s Quest

The Queen’s Quest is a 2D tile-based adventure game built with HTML5 Canvas and vanilla JavaScript.

Explore maze-like levels, collect all chests, and reach the King to progress to the next level. The game features overlay-based UI screens, a pause system, sound effects, and background music that changes per level.

# Controls
- Arrow Keys / WASD – Move  
- ENTER – Start, continue, next level  
- ESC – Pause / Resume  
- R – Restart (while paused)  
- M – Mute / Unmute music  

# Tech
- HTML5 Canvas  
- Vanilla JavaScript
